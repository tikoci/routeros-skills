# RouterOS Scripting Language

RouterOS scripting now has its own dedicated skill:
[`routeros-scripting`](../../routeros-scripting/SKILL.md).

Use that skill for `.rsc`, `/system/script`, scheduler/Netwatch `on-event`
snippets, script-safe `[find]` selectors, array/global/function gotchas, file
handling, and script policies. Its canonical syntax reference is:
[`routeros-scripting/references/syntax.md`](../../routeros-scripting/references/syntax.md).

This forwarding file stays here because `routeros-fundamentals` historically linked
to `references/scripting.md`.
